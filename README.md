##  区块链备忘录


### 1. 项目简介

通过智能合约，可将备忘录放在星云链上，这个备忘录具有不可篡改的特性

### 2. 项目使用方法

- 安装星云链Google插件钱包

安装方法：[https://github.com/ChengOrangeJu/WebExtensionWallet](https://github.com/ChengOrangeJu/WebExtensionWallet)

- 克隆代码

```
$ git clone https://github.com/jimmyzhu2017/memorandum.git
$ cd memorandum
$ npm install
$ npm start
```
